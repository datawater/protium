## Building

Clone the project and cd into it
```shell
$ git clone https://github.com/datawater/protium && cd protium
```

Build the project

```shell
# For release mode
$ cargo build --release
# Test
$ cargo test --release
$ cargo run --release

# For development builds
$ cargo build
# Test
$ cargo test
$ cargo run
```
